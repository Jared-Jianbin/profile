App Introduction
  • Hosted at Heroku
  • Backend: Used firebase as a fake server, meals data was fetched from it
  • Cart Function: Able to add items to cart and reflect changes, add or remove manipulation in the cart
  • Clicked order to enter user data, simple validation applies
  • Upon confirming user data, a post request with user info and ordered items will be sent to server ("write" rule at firebase has been disabled due to security concern)
Main Concepts
  • Basic react concepts including components, state management, lifting states up, and props drilling
  • Events Handling in React: onClick, onBlur, onChange, etc
  • React Lifecycles: mounting, updating, unmounting
  • Deep understand of React Hooks: useState, useRef, useEffect, useReduce, useContext, useCallback
  • Connected to server side with HTTP requests: get, post, put, delete
  • Form validation and custom hooks
